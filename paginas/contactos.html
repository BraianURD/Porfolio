Servicios

Las tecnologías que utilizó para el diseño y desarrollo de páginas web dependen del tipo de página web y requerimientos de los clientes, algunos de los lenguajes de programación que suelo utilizar son: HTML5, CSS, SASS Y GIT en todos mis desarrollos acompaño al cliente en cuatro etapas de desarrollo, las cuales son:

•Primer Etapa: Consiste en la primer a reunión con el cliente en la cual establecemos los requerimientos de diseño y funcionalidades, recolectamos información como textos, imágenes y demás recursos necesarios para el desarrollo.

•Segunda Etapa: Segunda reunión con el cliente para mostrar avances del proyecto y aprobación de diseño y estructura funcional.

•Tercer Etapa: Esta etapa se hace entrega del proyecto o página web al cliente con capacitación de administración si es el caso.

•Cuarta Etapa: Acompañamiento durante tres meses al cliente para garantizar el correcto funcionamiento.
